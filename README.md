# FocusFlow-Student-Productivity-Dashboard
A browser-based productivity dashboard designed to help students manage daily tasks, notes, time tracking, and real-time information within a single interactive interface.
---

## 🚀 Features

### ✅ Task Management System
- Add, edit, delete, and complete tasks
- Persistent storage using localStorage

### 📝 Notes Management
- Create, edit, and delete notes
- Notes saved automatically in browser storage

### 🌦 Weather Application
- Real-time weather updates using OpenWeather API
- Displays temperature, humidity, wind speed, sunrise, and sunset timings

### 💡 Quote Generator
- Fetches motivational quotes using Fetch API

### ⏱ Stopwatch / Productivity Timer
- Start, stop, and reset functionality

### 🌙 Dark Mode Toggle
- Persistent theme preference using localStorage

### 📱 Responsive UI
- Optimized for desktop and mobile devices
---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- REST APIs
- Fetch API
- localStorage
---

## ⚙️ Functional Modules

| Module | Functionality |
